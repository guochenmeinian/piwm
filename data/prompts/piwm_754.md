# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
27岁左右的女性，齐肩黑色直发，穿浅米色针织开衫和白色T恤，斜挎一个棕色小包；旁边同行朋友为同龄女性，短发，穿浅灰色卫衣，手里拿着手机
人物背景：周末和朋友一起逛商场便利区的年轻上班族，正在智能售货机前共同挑选一款饮品。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：她认为眼前这款商品看起来符合两人的口味和当下需求，朋友也表现出认可。
- Desire：她想在不花太多时间纠结的情况下，与朋友快速达成一致选择。
- Intention：她倾向用轻微点头和小幅手势向朋友做出柔和推荐，推动两人共同决定。

# Observable Performance
- 主要可见行为：两人站在设备前正面看向前方，视线在同一选择区域和彼此之间短暂切换，先后轻轻点头，主顾客抬起一只手做出小幅示意动作，像是在温和推荐同一个选择。
- 面部状态：主顾客表情放松，嘴角微微上扬，眼神专注但不纠结；朋友也露出轻微认可的微笑。
- 身体姿态：主顾客身体微微前倾，肩膀放松，双手保持在胸前到腰部范围内；朋友站在她侧后方半步，身体朝向同一方向，姿态自然开放。

# 10-Second Timeline
- 0–2 秒：27岁左右的女性和同龄朋友一起进入画面中央，正面站定后看向前方同一位置，双手自然露在画面内。
- 2–5 秒：主顾客微微前倾观察，眼神专注，朋友也靠近半步看向同一方向，两人的表情都显得感兴趣。
- 5–8 秒：主顾客轻轻点头并抬手做出一个小幅开放手势，像是在柔和推荐，朋友随即也点头表示认可。
- 8–10 秒：两人短暂对视后再次看向前方，主顾客保持轻微微笑，手势收回，呈现出接近共同决定的状态。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、夸张点头或明显舞台化动作
