# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
23岁左右的女性，长直发披肩，穿浅灰色短款针织开衫和白色T恤，斜挎米色小包；身旁朋友为同龄女性，扎低马尾，穿淡蓝色牛仔外套，手拿一部手机
人物背景：和朋友一起在便利区停下的年轻女性，正对某款带新功能的商品产生兴趣但还没完全理解用途。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：她认为这款商品的功能听起来新鲜，可能有用，但仅凭当前看到的信息还不够理解。
- Desire：她想快速弄清楚这个功能具体怎么使用、是否适合自己和朋友的需求。
- Intention：她倾向先和朋友确认理解，再等待或寻找一个简短演示来帮助判断。

# Observable Performance
- 主要可见行为：两人停在设备前，视线在前方同一区域与彼此之间来回切换，主顾客偶尔抬手做小幅指向和比划，朋友轻轻点头又微微摇头，表现出中等程度的犹豫与好奇。
- 面部状态：主顾客眉头轻微收紧，眼神专注，嘴角保持中性，偶尔露出疑惑的浅表情；朋友表情同样认真，带有好奇但不确定的神态。
- 身体姿态：主顾客身体正面朝向镜头，肩膀略向前倾，双手保持在胸前画面内；朋友站在她一侧半步内，同样面向镜头，姿态放松但注意力集中。

# 10-Second Timeline
- 0–2 秒：23岁左右的长发女性和扎低马尾的朋友一起停下，正面看向前方，两人的双手都自然停在画面内。
- 2–5 秒：主顾客微微前倾，抬起一只手在胸前小幅比划，视线从前方移向朋友，像是在确认自己对功能的理解。
- 5–8 秒：朋友轻轻点头后又露出一点迟疑，主顾客眉头轻皱，手指短暂悬停，表现出想了解更多但尚未决定的状态。
- 8–10 秒：两人重新看向前方，主顾客收回手并保持专注表情，像是在等待一个简短演示来说明功能。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、夸张点头或明显舞台化动作
