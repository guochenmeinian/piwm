# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
45岁左右的男性，短黑发略带灰白，穿浅灰色短袖衬衫和深色西裤，手腕戴黑色智能手表，左手拿着一部手机。
人物背景：午休后路过办公楼便利区的中年男性，想顺手买点东西但还没有明确需求。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：他认为眼前有几类可能合适的选择，但暂时判断不出哪一种更符合自己午休后的需要。
- Desire：他想快速找到一个合适的购买方向，避免在选择上花太久时间。
- Intention：他倾向继续扫视和比较，并需要一个关于口味、用途或预算的提示来帮助自己收拢注意力。

# Observable Performance
- 主要可见行为：他站在镜头前停留，视线在画面外的多个位置来回移动，偶尔低头看手机又抬头继续观察，手指轻轻摩挲手机边缘，动作克制但显出中等犹豫。
- 面部状态：表情平静略带迟疑，眉头轻微收紧，嘴唇自然闭合，眼神不断搜索但没有明显聚焦点。
- 身体姿态：身体正面朝向镜头，肩膀放松但略微前倾，双手保持在胸腹前方画面内，重心偶尔在两脚之间轻微转换。

# 10-Second Timeline
- 0–2 秒：这名45岁左右的男性从画面下方正面进入并停住，抬眼看向前方，左手握着手机停在腹前。
- 2–5 秒：他视线从左侧缓慢移到右侧，又短暂停在中间位置，眉头轻轻皱起，像是在比较不同类别的选择。
- 5–8 秒：他低头看了一眼手机，右手手指轻触手机边缘，随后再次抬头扫视前方多个位置，仍没有固定注视某一点。
- 8–10 秒：他身体微微前倾，视线停在中间偏上的区域片刻，表情仍然犹豫，像是在等待一个提示来帮助决定关注方向。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
