# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
40岁左右的男性，短黑发略带发际线后移，穿浅蓝色长袖衬衫和深灰色西裤，胸前挂着工牌，右手拿一部黑色手机
人物背景：午休后路过便利区的中年男职员，正在快速查看一款可能适合午后提神的商品。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：他认为眼前这款商品看起来符合午后提神和快速补充状态的需求。
- Desire：他想在不耽误返回工位的情况下，尽快确认这款商品是否值得购买。
- Intention：他倾向继续集中查看，并且很可能在温和推荐引导下进一步产生购买欲望。

# Observable Performance
- 主要可见行为：他在镜头前停下后视线持续集中在正前方略低处，停留时间稳定，右手握着手机但没有离开画面，左手轻轻抬起又放下，动作小而明确，犹豫很低。
- 面部状态：表情平静但专注，眉头轻微上扬，嘴角放松，眼神稳定，呈现出明显兴趣而非困惑。
- 身体姿态：身体正面朝向镜头，上半身微微前倾，双肩放松，双手都保持在画面内，站姿稳定且准备继续操作。

# 10-Second Timeline
- 0–2 秒：这名40岁左右的男性从画面前方停住，正面看向镜头下方，右手拿着黑色手机，表情由路过的自然状态转为专注。
- 2–5 秒：他保持视线集中，头部微微向前倾，眼睛在同一小范围内停留，像是在认真确认眼前选项的关键信息。
- 5–8 秒：他轻轻点了一下头，左手从身侧抬到胸前又缓慢放下，脸上保持平静但有兴趣的神情，没有明显迟疑。
- 8–10 秒：他继续正面站稳，目光仍然锁定前方略低处，右手手机保持在画面内，身体微微前倾，呈现出容易被温和推荐推进的状态。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
