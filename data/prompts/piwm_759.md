# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
23岁左右的女性，长直发披肩，穿浅米色针织开衫和白色内搭，搭配浅蓝色牛仔裤，斜挎一个小号棕色包，身旁有一位同龄女性朋友穿灰色卫衣共同看向前方
人物背景：和朋友一起逛商场便利区的年轻女性，停在智能售货设备前对饮料选择产生兴趣但还在犹豫。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：她认为面前有几款饮料都可能合适，但还不确定哪一款更值得选，朋友的看法会影响她的判断。
- Desire：她想找到一款自己和朋友都觉得不错、适合现在购买的饮料。
- Intention：她倾向先用眼神和手势询问朋友意见，再一起决定是否购买。

# Observable Performance
- 主要可见行为：她的目光在前方不同位置之间来回扫视，右手轻轻抬起做小幅指示动作，随后微微转向身旁朋友，用询问式眼神等待回应。
- 面部状态：表情专注但略带犹豫，眉头轻微收起，嘴角放松，转向朋友时眼神带有征询感。
- 身体姿态：身体正面朝向镜头并略微前倾，双手始终在画面内，一只手靠近斜挎包带，另一只手在胸前小幅比划，肩膀放松。

# 10-Second Timeline
- 0–2 秒：23岁左右的女性和身旁同龄女性朋友一起站定在镜头前，两人都正面看向前方，她的目光先快速扫过几个位置。
- 2–5 秒：她微微前倾，眼睛在左右两个方向之间来回比较，右手从包带旁抬起，在胸前做出轻轻指向的动作。
- 5–8 秒：她停顿片刻，眉头轻轻收起，视线从前方移向身旁朋友，像是在无声询问对方更倾向哪一个选择。
- 8–10 秒：她保持手在画面内，侧脸幅度很小地看向朋友并点头等待，朋友也继续看向前方，两人呈现共同决策前的中等犹豫状态。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
