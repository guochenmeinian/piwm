# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
20岁左右的男性，短黑发，穿浅灰色连帽卫衣和深色牛仔裤，肩背黑色双肩包，手里拿着手机和校园卡
人物背景：课间赶时间、路过校园便利区的男大学生，已经想买一款常喝饮料但还在确认是否值得花这笔钱。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：desire
- Belief：他认为这款常喝饮料能快速解渴提神，但当前价格可能略高，需要确认是否划算。
- Desire：他想在课间有限时间内买到熟悉的饮料，同时避免为一瓶饮料花得不值。
- Intention：他倾向继续盯着价格信息确认片刻，如果看到简短清晰的价格说明就会降低性价比顾虑并准备购买。

# Observable Performance
- 主要可见行为：他正面站在镜头前，视线在前方与略下方的同一区域之间来回移动，拇指轻轻摩擦手机边缘，偶尔在胸前轻微调整校园卡，表现出中等程度的犹豫。
- 面部状态：表情专注但略带迟疑，眉头轻微收紧，嘴唇自然抿住，眼神反复确认。
- 身体姿态：身体微微前倾，肩膀略紧，双手都在胸前画面内，手机和校园卡被握在一起，双脚基本站定但重心轻微左右切换。

# 10-Second Timeline
- 0–2 秒：20岁左右的男大学生正面已经停留在镜头前，黑色双肩包还挂在肩上，视线立刻落向前方略下的位置。
- 2–5 秒：他保持微微前倾，眼睛在前方和略下方反复扫视，右手拇指摩擦手机边缘，左手捏着校园卡停在胸前。
- 5–8 秒：他眉头轻轻皱起，像是在心里比较价格是否划算，校园卡在胸前被轻微调整，动作克制而犹豫。
- 8–10 秒：他再次看向略下方的同一区域，表情稍微放松但仍未完全决定，双手保持在画面内继续保持观察状态。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
