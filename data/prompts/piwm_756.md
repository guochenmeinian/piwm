# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
20岁左右的女性，黑色中长发自然披肩，穿浅灰色连帽卫衣和蓝色牛仔裤，肩背米白色帆布包，手里拿着一部手机和一本薄笔记本。
人物背景：课间路过自助零售区、还没想好买什么的女大学生。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：她觉得面前可能有适合课间快速补充精力的选择，但还不确定自己真正想要饮料、零食还是更清淡的东西。
- Desire：她想在短时间内找到一个合适又不太后悔的选择。
- Intention：她倾向于继续观察和比较，但需要被进一步询问或引导来明确自己最关注的是口味、饱腹感还是提神需求。

# Observable Performance
- 主要可见行为：她停在原地仔细看，视线在面前不同位置之间来回飘移，偶尔低头看手机又抬头，手指轻轻摩挲笔记本边缘，没有做出明确选择动作。
- 面部状态：表情认真但犹豫，眉头轻微收紧，嘴唇微抿，眼神有兴趣却缺少固定焦点。
- 身体姿态：身体正面朝向镜头，上半身微微前倾，双手都在胸前到腰间的画面范围内，肩膀略微收紧，站姿停顿感明显。

# 10-Second Timeline
- 0–2 秒：20岁左右的女大学生从画面前方停下，正面朝向镜头，上半身微微前倾，眼神先快速扫过面前区域。
- 2–5 秒：她把手机和薄笔记本拿在胸前，视线在几个不同方向之间缓慢游移，眉头轻轻皱起，表现出明显兴趣但没有锁定目标。
- 5–8 秒：她短暂低头看一眼手机，又抬头继续观察，手指摩挲笔记本边缘，嘴唇微抿，犹豫感加重。
- 8–10 秒：她仍停在原地，目光再次左右漂移，身体保持微微前倾，像是在等待进一步提示来明确自己真正想要什么。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、夸张点头或明显舞台化动作
