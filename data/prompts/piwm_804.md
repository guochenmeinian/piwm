# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
20岁左右的女性，黑色中长发自然披肩，穿浅灰色连帽卫衣和深色牛仔裤，肩背米白色帆布包，手里拿着一张校园卡和手机。
人物背景：课间路过便利区、想买一瓶提神饮料的女大学生。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：她认为眼前这款功能饮料大概率适合课间提神，但还需要确认主要成分、口味和适用场景是否符合自己的需求。
- Desire：她想先看清核心参数，判断这款饮料是否能在接下来的课程前提供合适的补充。
- Intention：她倾向于继续停留并查看属性信息，再决定是否进入购买操作。

# Observable Performance
- 主要可见行为：她站在原地正面朝向镜头，视线在镜头下方的小范围区域来回移动，偶尔轻轻点头，又短暂停顿，手指无意识地摩挲校园卡边缘。
- 面部状态：表情专注但略带犹豫，眉头轻微收紧，嘴唇自然抿住，眼神在思考和确认之间切换。
- 身体姿态：身体微微前倾，双肩放松，双手保持在胸前偏下位置，重心偶尔从一只脚轻轻转到另一只脚。

# 10-Second Timeline
- 0–2 秒：这名20岁左右的女大学生正面站定，微微前倾，视线落向镜头下方，像是刚注意到一款已基本锁定的功能饮料信息。
- 2–5 秒：她的眼睛缓慢左右扫动，眉头轻轻皱起，右手拇指摩挲着校园卡边缘，表现出正在确认主要成分的专注状态。
- 5–8 秒：她短暂抿唇并轻轻点了一下头，又把视线移回原处，像是在比较口味和适用场景是否适合接下来的课程。
- 8–10 秒：她保持正面姿态停留片刻，神情仍有中等程度的犹豫，双手没有离开画面，像是准备继续查看核心参数后再决定是否购买。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、夸张点头或明显舞台化动作
