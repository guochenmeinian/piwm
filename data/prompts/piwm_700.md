# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
一名 25–35 岁的普通亚洲女性，穿着简洁自然的日常休闲服饰，外表生活化，气质自然克制。
人物背景：双休日无聊出来逛逛街的普通消费者。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：售货机里可能有她感兴趣的商品，值得停下来仔细看一看。她认为当前还需要继续确认和比较信息，暂时没有足够把握立即购买。
- Desire：想进一步了解商品是否值得购买，并确认它是否符合自己的需求。
- Intention：继续停留观察，反复确认信息并进行比较，在做出购买决定前保持谨慎。

# Observable Performance
- 主要可见行为：顾客走近后在机器前停留，身体轻微前倾，视线先稳定看向前方，再短暂下移，并在前方不同位置之间自然切换数次，持续表现出浏览、比较和确认信息的状态。
- 面部状态：表情自然克制，带有轻微思考、权衡和犹豫，没有夸张惊讶或明显情绪波动。
- 身体姿态：正面朝向机器，站姿稳定，停留在原地，身体轻微前倾，双手自然放置并始终保持在画面内。

# 10-Second Timeline
- 0–2 秒：顾客从前方走近，进入镜头中心，脚步逐渐放慢，在机器前停下，并将注意力集中到机器前方开始认真观察。
- 2–5 秒：顾客继续停留，身体轻微前倾，目光稳定看向前方，随后短暂下移视线，再回到前方，像是在浏览和确认不同信息。
- 5–8 秒：顾客头部和目光在前方不同位置之间自然切换数次，节奏缓慢真实，表现出比较、思考和犹豫。
- 8–10 秒：顾客继续停留在原地，再次短暂下移视线，然后回到前方，保持思考和犹豫状态，没有支付、没有扫码、没有离开。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
