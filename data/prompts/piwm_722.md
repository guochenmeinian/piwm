# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
26岁左右的女性，黑色中长发低马尾，穿浅灰色西装外套和白色衬衫，搭配黑色直筒裤，肩上背一个深棕色通勤包
人物背景：下班路过便利区的年轻白领女性，被前方亮光短暂吸引但并未决定停留。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：attention
- Belief：她注意到前方有醒目的亮光和可能与自己相关的购买信息，但还不确定是否值得停下来查看。
- Desire：她想快速判断这里是否有值得关注的商品，同时不耽误继续赶路。
- Intention：她倾向于只用几秒钟扫视一下，如果没有立刻吸引她的理由就继续离开。

# Observable Performance
- 主要可见行为：她从通道方向缓步经过时被亮光吸引，脚步明显放慢但没有真正停稳，目光在前方区域短暂左右扫视，双臂始终自然下垂，没有伸手或操作动作。
- 面部状态：表情克制而犹豫，眼神短暂聚焦后又游移，眉头轻微收紧，嘴唇自然闭合。
- 身体姿态：身体正面朝向镜头但重心略偏向通道方向，肩膀放松，双臂垂在身体两侧，姿态保持观望和随时继续走开的状态。

# 10-Second Timeline
- 0–2 秒：这位26岁左右的女性从通道方向缓慢经过，前方亮光让她的视线抬起，步伐开始放慢但双臂仍自然下垂。
- 2–5 秒：她保持正面朝向镜头，眼睛在前方区域左右扫视约两秒，身体重心仍略微偏向通道方向，没有完全站定。
- 5–8 秒：她的眉头轻轻收紧，嘴唇闭合，脚尖和肩部微微朝离开的方向偏转，表现出明显犹豫但没有伸手动作。
- 8–10 秒：她最后短暂看向前方中央，随后视线移开，身体继续保持可离开的姿态，整体仍处于安静观望状态。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
