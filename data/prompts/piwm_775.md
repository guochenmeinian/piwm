# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
28岁左右的男性，短黑发，穿浅灰色连帽卫衣和深色休闲裤，斜挎一个黑色通勤包，身旁站着一位年龄相近的女性朋友，扎低马尾，穿米色针织开衫，手里拿着手机。
人物背景：午休间隙和朋友一起路过便利区、想快速买点冷饮的年轻上班族。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：desire
- Belief：他认为眼前的选择很适合现在购买，但担心午休时间不多，怕耽误接下来的安排。
- Desire：他想和朋友一起尽快做出轻松的购买决定，同时希望确认还有足够时间不用着急。
- Intention：他倾向于先用眼神和朋友确认彼此意见，再放慢一点动作，准备安心完成选择。

# Observable Performance
- 主要可见行为：他和朋友都正面站在镜头前，视线在前方与彼此之间来回切换，偶尔低头看一眼手机时间，手指轻轻摩挲包带，动作有短暂停顿但没有明显退缩。
- 面部状态：表情带着想买的兴趣和中等程度的犹豫，眉头轻微收紧，随后在看到朋友神情放松后自己也稍微缓和下来。
- 身体姿态：身体微微前倾，双肩略紧，双手始终在画面内；朋友站在他侧后方半步，也保持前倾观察，两人之间通过短暂对视传递时间压力。

# 10-Second Timeline
- 0–2 秒：28岁左右的男性和身旁的女性朋友一起走近并停在镜头前，两人都面向前方，男性的手扶着黑色斜挎包带。
- 2–5 秒：男性目光在前方停留片刻后转向朋友，朋友也看向他并轻轻抬起手机，两人的表情显示出想买但担心时间不够。
- 5–8 秒：男性低头快速扫一眼手机时间，又抬头看向前方，手指在包带上轻轻摩挲，朋友的神情略微放松，像是在示意不用太急。
- 8–10 秒：男性的眉头放松一些，和朋友再次短暂对视后一起把视线转回前方，身体仍微微前倾，像是准备更从容地做出购买决定。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
