# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
45岁左右的男性，短黑发略带灰白，穿浅蓝色衬衫和深灰色西裤，领口微松，左手戴腕表，右手拿着手机，肩上斜挎一个黑色通勤包
人物背景：午休期间路过办公楼便利区的中年男职员，已经想买一瓶饮品但担心返回会议来不及。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：desire
- Belief：他认为眼前的商品基本符合自己的午休需求，但购买过程是否足够快仍让他有些不确定。
- Desire：他想尽快完成购买，在不耽误午休后续安排和会议时间的情况下拿到饮品。
- Intention：他倾向于继续购买，但会先反复确认手机时间，再决定是否立刻操作。

# Observable Performance
- 主要可见行为：他站在设备前停留，视线在前方和手机之间来回切换，右手把手机举在胸前低头查看，左手手指轻轻敲动包带，动作克制但带有时间压力。
- 面部状态：眉头轻微皱起，嘴唇抿住，眼神专注又有些犹豫，整体表现为想买但担心耗时的中等焦虑。
- 身体姿态：身体正面朝向镜头，上半身略微前倾，双肩有些紧绷，双手始终在画面内，重心偶尔在两脚之间小幅转换。

# 10-Second Timeline
- 0–2 秒：45岁左右的男性正面站到镜头前，先看向前方，随后快速低头看一眼胸前的手机。
- 2–5 秒：他保持身体朝前，右手握着手机不遮挡面部，左手轻敲包带，眉头微皱地在前方和手机之间切换视线。
- 5–8 秒：他轻轻吸一口气，上半身向前靠近一点，像是在确认自己已经决定购买，但又再次低头查看时间。
- 8–10 秒：他抬起头看向前方，表情稍微放松，左手停止敲动，身体稳定下来，呈现出被时间可控感安抚后准备继续操作的状态。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
