# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
24岁左右的女性，长直发披肩，穿浅米色针织开衫和白色内搭，斜挎一个棕色小包；身旁朋友为同龄女性，短发，穿浅蓝色牛仔外套，背一个黑色双肩包
人物背景：和朋友一起逛便利区的年轻女性，正在智能售货设备前比较两款候选饮品。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：她认为眼前有两款饮品都可能合适，但差异还不够明确，需要进一步比较关键信息。
- Desire：她想和朋友快速判断哪一款更符合当下口味和需求。
- Intention：她倾向于先通过对比信息缩小选择范围，再决定是否继续购买。

# Observable Performance
- 主要可见行为：她和朋友并排站在画面前方，视线在不同方向之间来回移动，各自用手指向不同侧的位置，随后短暂停顿并相互看一眼，表现出中等程度的犹豫和比较。
- 面部状态：她眉头轻微收紧，嘴唇微抿，表情专注而犹豫；朋友表情认真，带有轻微询问感。
- 身体姿态：她身体正面朝向镜头，上半身微微前倾，双手保持在画面内，一只手指向左侧后收回；朋友同样正面朝向镜头，稍微偏头并抬手指向右侧。

# 10-Second Timeline
- 0–2 秒：24岁左右的长发女性和短发朋友并排进入画面中心，正面停下，双手自然垂在身前，目光看向前方不同位置。
- 2–5 秒：长发女性微微前倾，抬起右手指向左侧位置，朋友同时抬手指向右侧位置，两人无声地交换眼神。
- 5–8 秒：长发女性收回手，视线在左侧和右侧之间来回移动，眉头轻皱，朋友也短暂低头思考后再次看向前方。
- 8–10 秒：两人保持正面站姿，长发女性轻轻点头又停住，朋友放下手，双方仍带着比较和犹豫的神情看向前方。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、夸张点头或明显舞台化动作
