# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
55 岁左右的女性，短卷发，穿浅灰色针织开衫和米色长裤，手腕戴一只细表，双手拿着一个棕色小手提包
人物背景：午休后路过办公楼便利区的中老年女性顾客，想顺手买一份合适的饮品但还在犹豫。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：desire
- Belief：她觉得眼前的选择里应该有适合午休后饮用的商品，但不确定哪一个更符合自己的口味和身体需求。
- Desire：她想买一份清爽又稳妥的饮品，同时希望得到一个温和的推荐来增强选择信心。
- Intention：她倾向于在得到明确但不强硬的推荐后，选择其中一个并继续购买。

# Observable Performance
- 主要可见行为：她站在原地反复看向前方偏下的位置，视线在不同选项间缓慢移动，手指轻轻摩挲包带，偶尔微微抬手又放下，表现出中等程度的犹豫。
- 面部状态：表情温和但略带迟疑，眉头轻轻收起，嘴唇微抿，偶尔露出很浅的确认式点头表情。
- 身体姿态：身体正面朝向镜头，肩膀放松但上身微微前倾，双手始终在胸前或腰前握着手提包，重心轻微左右转移。

# 10-Second Timeline
- 0–2 秒：55 岁左右的女性从画面前方停稳，正面朝向镜头，上身微微前倾，目光落在前方偏下的位置。
- 2–5 秒：她的视线缓慢从左侧移到右侧，双手握着棕色小手提包，手指轻轻摩挲包带，脸上带着谨慎的思考神情。
- 5–8 秒：她微微抬起一只手像是准备确认选择，又停在半空后放回包带上，眉头轻轻收起，表现出想买但还不完全确定。
- 8–10 秒：她轻轻点了一下头，表情稍微放松，目光重新稳定在前方偏下的位置，像是在等待一个温和推荐后再做决定。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
