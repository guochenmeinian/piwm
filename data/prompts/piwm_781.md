# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
28岁左右的男性，短黑发，穿浅蓝色衬衫和深灰色西裤，胸前挂工牌，左手戴智能手表，右手拿着手机和一张办公门禁卡
人物背景：午休时来到办公楼便利区的年轻白领，想快速买一件饮品却在最后决定前反复犹豫。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：desire
- Belief：他认为自己已经选中了合适的饮品，但又担心现在购买不够理性或之后会后悔。
- Desire：他想在午休时间内快速确认选择，降低决策压力，并安心完成这次购买。
- Intention：他倾向再次确认自己的选择是否没问题，如果获得足够确定感就立刻继续下一步。

# Observable Performance
- 主要可见行为：他反复把右手手机在胸前轻微调整，视线在正前方小范围左右移动，手指轻敲手机边缘，几次像要确认又停住。
- 面部状态：眉头轻微皱起，嘴唇抿住，眼神专注但犹疑，偶尔短暂停顿后又露出不确定的神情。
- 身体姿态：身体正面朝向镜头，上半身微微前倾，双肩略紧，双手始终在胸前到腰部之间活动，重心在两脚之间轻微来回转移。

# 10-Second Timeline
- 0–2 秒：这名28岁左右的男性白领正面站定，右手握着手机，抬眼看向前方，眉头微皱像是已经锁定了一个选择。
- 2–5 秒：他把手机向前抬了几厘米，拇指悬停片刻，又慢慢收回，左手捏着门禁卡轻轻摩挲，视线在正前方小范围左右扫动。
- 5–8 秒：他短暂停住，像是在说服自己，但随即抿紧嘴唇，身体后退半步又重新前倾，右手再次抬起后停住。
- 8–10 秒：他深吸一口气，肩膀稍微放松，目光重新集中在正前方，手机稳定在胸前，表现出等待最后确认后继续操作的状态。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
