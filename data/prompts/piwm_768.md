# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
45岁左右的女性，短款微卷黑发，穿米色针织开衫和深绿色连衣裙，手腕戴细表，肩背一个棕色皮质小挎包，双手自然放在身前
人物背景：逛街路过商场便利区的中年女性，经过多轮比较后已经基本确定想买的饮品，只差最后确认。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：desire
- Belief：她认为自己已经比较清楚不同选择的差异，并且心里选定的那一款更符合当下逛街后想补充饮品的需求。
- Desire：她希望快速确认这个选择是合适的，并尽快完成购买，不再继续浪费时间比较。
- Intention：她倾向接受一个明确而果断的推荐，然后立即进入购买操作。

# Observable Performance
- 主要可见行为：她正面看向设备方向，视线稳定，偶尔轻微上下扫视，手指在身前短暂并拢后放松，几乎没有来回张望或反复纠结的动作。
- 面部状态：神情坚定，嘴唇轻轻抿住，眉眼放松但目光明确，表现出低犹豫和已做决定的状态。
- 身体姿态：身体站直并略微前倾，双肩平稳，双手始终在画面内靠近腰前，整体姿态稳定而准备行动。

# 10-Second Timeline
- 0–2 秒：45岁左右的女性正面站在镜头前，米色针织开衫和棕色小挎包清晰可见，她的目光稳定地落在设备方向。
- 2–5 秒：她轻微上下扫视，像是在最后确认选项，双手在腰前自然并拢，表情保持冷静坚定。
- 5–8 秒：她几乎不再左右比较，只是微微点头，嘴唇轻抿，身体略向前倾，显示出已经形成明确偏好。
- 8–10 秒：她抬起一只手停在身前准备操作，目光仍然坚定地看向设备方向，呈现出即将接受果断推荐并购买的状态。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、夸张点头或明显舞台化动作
