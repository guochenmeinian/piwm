# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
24岁左右的女性，齐肩黑色直发，穿浅灰色针织开衫和白色T恤，斜挎一个米色小包，手里拿着一部手机
人物背景：和同伴一起路过办公楼便利区的年轻女性，已经基本选好想买的饮品但在付款前想确认同伴是否同意。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：action
- Belief：她认为自己已经选定了合适的商品并可以直接付款，但同伴的意见可能影响最终决定。
- Desire：她想快速完成购买，同时避免买到同伴不想一起喝或不认可的选择。
- Intention：她准备支付前先停下，用眼神和轻微手势向身旁同伴征求确认，再继续完成付款动作。

# Observable Performance
- 主要可见行为：她一只手握着手机停在胸前，另一只手轻轻点了点手机边缘，随后短暂看向画面侧方像是在等待同伴回应，得到默许后重新把注意力收回前方并准备继续操作。
- 面部状态：表情克制而略带犹豫，眉头轻微收紧，嘴唇抿住；在获得同伴意见后神情放松，露出很浅的确认感。
- 身体姿态：身体正面朝向镜头，上半身微微前倾，肩膀略收，等待时头部小幅侧偏，最后重新站稳并把手机保持在可操作位置。

# 10-Second Timeline
- 0–2 秒：24岁左右的女性正面站在镜头前，手机握在胸前，视线集中在前方，像是已经进入付款前的最后确认。
- 2–5 秒：她手指悬在手机屏幕上方却没有立刻按下，眉头轻轻皱起，眼神短暂在前方和侧方之间移动。
- 5–8 秒：她保持身体朝前，只把头微微偏向画面侧方，轻轻抬了抬手机和眉毛，像是在无声询问同伴是否可以这样买。
- 8–10 秒：她看到侧方回应后表情放松，视线回到前方，手指重新落向手机，准备继续完成支付。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 允许出现与当前状态一致的轻微手机支付、刷卡、确认、等待出货或取货动作；动作应克制、真实，不要夸张贴屏、反复敲击或展示界面细节
- 不展示机器主动响应或屏幕 UI 内容；即使发生支付、刷卡、等待出货或取货，也只表现顾客自然动作，不展示二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、夸张点头或明显舞台化动作
