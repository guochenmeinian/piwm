# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
26岁左右的女性，齐肩微卷黑发，穿米白色针织开衫和浅蓝色牛仔裤，斜挎一个棕色小包，手里拿着手机和细长钱包
人物背景：和朋友一起逛商场便利区的年轻女性顾客，自己已经想买一份饮品但担心朋友觉得不划算。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：desire
- Belief：她认为眼前的选择很适合自己现在想喝点东西的需求，但价格或必要性可能会被同行朋友质疑。
- Desire：她想买下自己喜欢的那一款，同时希望确认这个决定是合理的、不需要因为朋友看法而退缩。
- Intention：她倾向于再短暂确认一次后下定决心购买，但需要获得一种被认可和被鼓励的自信感。

# Observable Performance
- 主要可见行为：她反复在正前方小范围扫视，手指轻轻摩挲钱包边缘，偶尔把手机抬起又放低，像是在比较后仍迟迟没有做出最终动作。
- 面部状态：眉头微微收紧，嘴唇轻抿，眼神专注又犹豫，偶尔露出一点想买但不够坚定的迟疑表情。
- 身体姿态：身体正面朝向镜头，上半身略微前倾，双肩有些收紧，双手始终停在胸前到腰部之间，姿态克制而不安定。

# 10-Second Timeline
- 0–2 秒：这位26岁左右的女性正面站定，微微前倾看向前方，双手握着手机和钱包，眼神很快被眼前的选择吸引。
- 2–5 秒：她的视线在正前方左右小幅移动，手指反复摩挲钱包边缘，眉头轻轻皱起，像是在衡量自己是否真的应该买。
- 5–8 秒：她短暂抬起手机又放低，嘴唇轻抿，眼睛向侧前方轻轻一瞥后立刻回到正前方，表现出担心同行朋友评价的犹豫。
- 8–10 秒：她深吸一口气，肩膀稍微放松，双手把钱包握得更稳，表情从焦虑转为略微坚定，像是正在被说服去相信自己的决定。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
