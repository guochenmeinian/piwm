# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
27岁左右的女性，齐肩黑色直发，穿米白色针织开衫和浅蓝色衬衫，下身深灰色直筒裤，单肩背一个棕色通勤包，双手自然露在画面内
人物背景：逛街路过便利零售区的年轻白领，已经在智能售货设备前停留了一会儿，对某款商品产生兴趣但还没下定决心。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：她认为眼前这款商品可能适合现在的需求，但还不确定是否比其他选择更合适。
- Desire：她想快速确认这款商品是否值得购买，最好得到一个不打扰的轻度推荐来降低选择成本。
- Intention：她倾向于继续盯着这款商品观察几秒，如果出现温和提示或理由充分的推荐，就可能进一步做决定。

# Observable Performance
- 主要可见行为：她的视线始终集中在前方偏下的同一位置，偶尔轻微眨眼，右手指尖轻捏左手手背，动作克制，表现出中等程度的犹豫和比较。
- 面部状态：表情安静专注，眉头轻微收拢，嘴唇自然闭合，带有思考和不确定感但没有明显焦虑。
- 身体姿态：身体正面朝向镜头，肩膀放松但略向前倾，重心在两脚之间轻微变化，双手停在胸前到腰部之间的画面范围内。

# 10-Second Timeline
- 0–2 秒：这位27岁左右的女性白领正面站在镜头前，视线固定在前方偏下的位置，双手自然停在身前。
- 2–5 秒：她保持目光不移，右手指尖轻轻捏住左手手背，眉头微微收紧，像是在认真权衡眼前的选择。
- 5–8 秒：她身体略向前倾半分，重心轻轻换到另一只脚上，表情依旧专注而犹豫，似乎等待一个更明确但不强迫的推荐理由。
- 8–10 秒：她缓慢眨眼后仍看向同一位置，双手放松了一些，脸上的犹豫略有缓和，但还没有做出最终动作。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
