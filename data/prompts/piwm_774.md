# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
27岁左右的女性，黑色中长发低马尾，穿浅灰色西装外套和白色衬衫，搭配黑色直筒裤，左肩背米色通勤包，右手拿着手机和门禁卡。
人物背景：赶时间去开会、路过办公楼便利区的年轻白领女性，对一款想买的商品感兴趣但担心购买流程耽误行程。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：desire
- Belief：她认为这款商品正符合自己现在的需求，也相信自己会喜欢，但不确定购买和取货是否足够快。
- Desire：她想立刻买下这款商品，同时希望确认不会因为操作或等待而耽误接下来的行程。
- Intention：她倾向于在确认流程足够迅速后马上完成购买，否则可能暂时放弃离开。

# Observable Performance
- 主要可见行为：她多次在正前方视线范围内来回扫视，右手握着手机微微抬起又放低，左手轻捏包带，期间短暂低头看时间后又重新看向前方。
- 面部状态：眉头轻皱，嘴唇微抿，眼神专注但带明显犹豫，偶尔露出想下决定前的迟疑神情。
- 身体姿态：身体正面朝向镜头，肩膀略向前倾，一只脚轻微前后调整重心，整体姿态紧绷而急促。

# 10-Second Timeline
- 0–2 秒：这位27岁左右的女性白领已经停留在镜头前，正面抬眼看向前方，右手握着手机，左手抓住肩上的米色通勤包带。
- 2–5 秒：她的视线在正前方小范围来回移动，眉头轻轻皱起，右手把手机抬到胸前像是准备操作，却又停顿下来。
- 5–8 秒：她低头快速看了一眼手机时间，嘴唇抿紧，身体微微前倾，脚尖不安地调整位置，表现出想买但担心耽误时间的犹豫。
- 8–10 秒：她重新看向前方，深吸一口气，右手更稳定地举起手机靠近胸前，表情稍微放松，像是在被确认很快后继续保持观察状态。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、夸张点头或明显舞台化动作
