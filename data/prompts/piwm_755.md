# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
58岁左右的女性，短卷发夹杂少量灰白发，穿酒红色针织开衫和米色长裤，肩上挎一个深棕色布质手提包
人物背景：专程来到室内零售区逛智能售货设备的中老年女性顾客，停下来想看看有没有适合自己的商品。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：她认为眼前设备里应该有不少可选商品，可能有适合自己的东西，但暂时分不清该看哪一类。
- Desire：她想找到一个符合自己当下需求的商品方向，但还没有明确想买的类型。
- Intention：她倾向于继续观察并寻求一点提示或帮助，以便把选择范围聚焦下来。

# Observable Performance
- 主要可见行为：她停在画面中央，视线在不同方向来回扫视，偶尔短暂停住又移开，双手始终在胸前或腹前可见，一只手轻捏包带，另一只手小幅抬起又放下，没有立即做出购买动作。
- 面部状态：表情带有温和好奇和中等程度的犹豫，眉头微微收紧，嘴唇轻抿，眼神不断寻找重点。
- 身体姿态：身体正面朝向镜头，站姿稳定但略微前倾，肩膀放松，头部随着视线小幅左右转动，呈现认真查看但尚未决定的状态。

# 10-Second Timeline
- 0–2 秒：这位58岁左右的女性走到画面中央停下，正面朝向镜头，双手自然放在腹前，目光先从一侧缓慢扫向另一侧。
- 2–5 秒：她微微前倾，眼睛在不同方向来回查看，右手轻轻捏住深棕色手提包的包带，眉头略微收紧。
- 5–8 秒：她像是看到几个可能的选择，视线短暂停住又移开，左手小幅抬到胸前后又放下，表情显得拿不定主意。
- 8–10 秒：她重新看向镜头方向，嘴唇轻抿，保持正面站姿，神情像是在等待进一步提示来帮助自己确定想要的类型。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、夸张点头或明显舞台化动作
