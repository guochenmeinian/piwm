# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
45岁左右的男性，短黑发略带灰，穿浅灰色短袖POLO衫和深色休闲裤，左肩斜背一个黑色通勤包，右手拿着手机但不遮挡面部
人物背景：午休后路过写字楼便利区的中年男职员，停在智能售货设备前想弄清某款商品是否适合自己使用。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：他认为这款商品可能有用，但对具体使用场景和操作方式还不太了解。
- Desire：他想快速看到直观演示，以判断这款商品是否符合自己午后办公或日常使用需求。
- Intention：他倾向继续停留观察，并寻找能说明使用方法的演示信息，而不是立刻下单。

# Observable Performance
- 主要可见行为：他在镜头前停留不走，视线在前方不同位置缓慢移动，偶尔低头看一眼手机又抬头，手指轻轻摩挲手机边缘，动作克制但显示出中等犹豫。
- 面部状态：眉头微皱，眼神专注又带疑问，嘴唇轻抿，表情像是在等待更清楚的说明。
- 身体姿态：身体正面朝向镜头，肩膀略向前倾，双脚站稳，左手扶着包带，右手持手机停在胸前高度。

# 10-Second Timeline
- 0–2 秒：这名45岁左右的男性从画面下方走近并停住，正面朝向镜头，午休后的神情略显放松但注意力被前方内容吸引。
- 2–5 秒：他微微前倾，视线从左到右扫过前方，眉头轻皱，右手握着手机停在胸前，像是在努力理解商品的用途。
- 5–8 秒：他低头快速看一眼手机又抬头，左手轻拉包带，嘴唇抿起，脸上露出需要进一步演示说明的疑问感。
- 8–10 秒：他没有离开，而是继续站在原地凝视前方，身体保持轻微前倾，表现出仍在兴趣阶段、准备继续观看演示的犹豫状态。

# Motion Quality
- 允许顾客从画面边缘自然走入或放慢脚步，但动作必须平稳，不要突然闯入、骤停或立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、夸张点头或明显舞台化动作
