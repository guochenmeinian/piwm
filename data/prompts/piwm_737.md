# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
45岁左右的男性，短黑发略带灰白，穿浅灰色短袖衬衫和深色休闲裤，手腕戴黑色手表，单肩背一个深棕色公文包
人物背景：午休后路过办公楼便利区的中年男性，想买一件合适的小商品但正在认真比较价格和规格。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：interest
- Belief：他认为眼前有几款商品都可能符合需求，但价格、容量和规格信息比较零散，暂时难以判断哪一款更划算。
- Desire：他想快速整理出几款商品之间的差异，找到性价比和实际需求都合适的选择。
- Intention：他倾向于继续停在原地来回查看，并优先获取更清晰的价格与规格对比信息后再决定。

# Observable Performance
- 主要可见行为：他站在原地不移动，视线在几个方向之间反复切换，偶尔微微抬手像是在心里比对选项，手指轻敲公文包肩带但没有做出选择动作。
- 面部状态：眉头轻皱，嘴唇微抿，眼神专注但带有明显犹豫，像是在努力整理零散信息。
- 身体姿态：身体正面朝向镜头，上半身略微前倾，双肩稍紧，双手始终在胸前到腰部范围内，一只手扶着肩带，另一只手偶尔半抬起又放下。

# 10-Second Timeline
- 0–2 秒：45岁左右的男性正面站定，午休后的神情略显疲惫，视线先向画面左下方停留片刻，像是注意到一个选项。
- 2–5 秒：他把视线缓慢移到画面右下方，眉头轻皱，右手半抬到胸前又停住，表现出正在比较另一组价格和规格。
- 5–8 秒：他仍站在原地，目光在左右之间来回切换两次，手指轻轻敲着公文包肩带，脸上露出信息不够集中、难以下判断的犹豫感。
- 8–10 秒：他微微前倾，眼睛重新扫向中间偏下的位置，嘴唇抿紧，像是准备先弄清几款商品的对比信息再继续决定。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 手部不触碰画面外不存在的具体商品或按钮，不做扫码、付款、刷卡、取货等已交互动作；若手里拿着手机、卡片或钱包，只能自然握持或在胸前轻微调整，不能贴向屏幕
- 不展示机器主动响应、屏幕 UI 内容、二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、重复或明显舞台化的头部动作
