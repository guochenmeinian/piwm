# Goal
生成一段 10 秒、写实自然、单镜头的视频。视频内容是智能零售设备前置摄像头捕捉到的顾客当前状态片段。

# Camera And Framing
- 视角：设备正面上方前置摄像头，朝外拍摄顾客
- 镜头：固定机位，轻微俯视，不平移，不旋转，不推拉，不变焦
- 构图：顾客位于画面中央附近，头部、面部、上半身、双手持续清晰可见
- 画面：真实商用摄像头质感，光线稳定，无遮挡，无镜头漂移
- 禁止出现机器本体、机器边框、商品、货架、屏幕 UI、价格标签、文字叠层

# Subject Continuity
- 顾客外观在 10 秒内完全一致，不换脸、不换衣、不增减配饰
- 只有一名顾客，无其他人进入画面
- 动作幅度克制，表情自然，不做戏剧化表演
- 顾客的犹豫应表现为细微确认、短暂停顿、轻度迟疑，不要演成明显困惑、茫然失措或夸张纠结

# Character
20岁左右的男性，短黑发，穿浅灰色连帽卫衣和深色牛仔裤，背一个黑色双肩包，手里拿着一部手机
人物背景：课间匆忙赶来便利区买饮料的男大学生，准备付款时担心上课时间快到了。

# Internal State Reference
这些状态只用于约束行为气质，不应直接以文字、符号或 UI 出现在画面里。
- 购买阶段：action
- Belief：他认为自己已经选好想买的东西，只差完成支付，但担心付款过程会耽误赶回教室的时间。
- Desire：他想尽快确认支付并离开，同时避免因为匆忙而点错或操作失误。
- Intention：他倾向于继续完成支付，但在最后确认前会短暂停顿并再次确认时间是否来得及。

# Observable Performance
- 主要可见行为：他双手都在画面内，一手握着手机，另一只手的拇指在手机边缘轻轻停住，视线在前方和手机之间快速切换，动作有短暂停顿但没有后退。
- 面部状态：眉头轻微皱起，嘴唇抿住，眼神带着赶时间的紧张感和最后确认前的迟疑。
- 身体姿态：身体正面朝向镜头，肩膀略微前倾，背包带压在双肩上，站姿稳定但重心轻轻向一侧移动。

# 10-Second Timeline
- 0–2 秒：20岁左右的男性正面站定，低头看了一眼手里的手机，又迅速抬眼看向前方，表情显得有些赶时间。
- 2–5 秒：他用拇指在手机边缘轻轻滑动，肩膀微微前倾，眼神在手机和前方之间来回确认。
- 5–8 秒：他准备按下确认时突然停住半秒，嘴唇抿紧，眉头轻皱，像是在判断这一步会不会耽误上课。
- 8–10 秒：他深吸一口气后放松一点，拇指重新靠近手机操作位置，身体仍面向前方，表现出准备继续完成支付的状态。

# Motion Quality
- 视频开始时顾客应已经处在摄像头可见范围内，默认是已停留并正在观察的片段，不要拍成从画外突然闯入、骤停、再立刻表情大变
- 所有头部、视线、停顿、前倾、手部变化都应缓慢、连续、符合真实顾客行为
- 视线变化要自然，可短暂停留后再移动，不能高频乱跳
- 允许出现与当前状态一致的轻微手机支付、刷卡、确认、等待出货或取货动作；动作应克制、真实，不要夸张贴屏、反复敲击或展示界面细节
- 不展示机器主动响应或屏幕 UI 内容；即使发生支付、刷卡、等待出货或取货，也只表现顾客自然动作，不展示二维码、付款界面、价格文字或终端弹窗

# Negative Constraints
- 不出现售货机本体、柜门、商品、货架、二维码、付款界面、促销字幕
- 不出现镜头抖动、剪辑、转场、景别变化、夸张景深变化
- 不出现畸形手部、多余手指、面部扭曲、身体比例突变
- 不出现突然转身离开、突然贴近镜头、夸张点头或明显舞台化动作
