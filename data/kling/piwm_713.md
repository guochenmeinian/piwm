# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
26岁左右的女性，齐肩黑色直发，穿浅灰色针织开衫和白色T恤，下身为深色直筒裤，肩上背一个米色帆布托特包。
背景：午休后路过办公楼便利区的年轻女白领，已经看中智能售货机里的一款饮品，准备快速购买。

# State
- 购买阶段：desire
- Belief：她认为眼前这款饮品正好符合自己想要提神解渴的需求，选择基本已经明确。
- Desire：她想尽快买到机器里的那款产品，减少继续比较的时间。
- Intention：她准备拿出手机进行扫码或支付，进入购买动作。

# Behavior
- 可见行为：她起初短暂看向不同位置，随后目光固定在一个方向不再扫视，右手从裤子口袋里掏出手机并抬到胸前。
- 表情：表情从轻微思考和抿唇，逐渐转为平静确定，眉头放松，眼神集中。
- 姿态：身体正面朝向镜头，站姿稳定，上半身微微前倾，一只手扶着包带，另一只手在画面内从口袋取出手机。

# Timeline
- 0–2 秒：这位26岁左右的女性正面站在镜头前，眼神在前方轻微移动，像是在最后确认选择。
- 2–5 秒：她的视线逐渐停在固定方向，嘴唇轻轻抿了一下，脸上的思考感开始减弱。
- 5–8 秒：她眉头放松，表情变得确定，右手伸向裤子口袋并把手机掏出来。
- 8–10 秒：她将手机拿到胸前，目光仍固定在前方，身体微微前倾，呈现准备购买的状态。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
