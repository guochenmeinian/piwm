# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
22岁左右的女性，黑色中长发自然披肩，穿浅灰色连帽卫衣和浅蓝色牛仔裤，肩背一个米白色帆布包，手里拿着手机
背景：午后课间路过自助零售区、想买一瓶饮品的年轻女大学生

# State
- 购买阶段：interest
- Belief：她认为眼前可选的饮品看起来比较符合自己现在想喝点清爽饮料的需求。
- Desire：她想快速挑出一瓶口味合适、喝起来轻松的饮品。
- Intention：她倾向于继续对比两三个选项后很快做出选择。

# Behavior
- 可见行为：她的目光在不同位置之间来回停留，右手握着手机，左手轻轻抬起又放下，像是在对比选项但没有明显纠结。
- 表情：表情放松，眼神专注，嘴角轻微上扬，带着明确的兴趣和轻微期待。
- 姿态：身体正面朝向镜头并微微前倾，肩膀自然放松，双手始终在画面内。

# Timeline
- 0–2 秒：22岁左右的女性走近后停在画面中央，身体微微前倾，视线先落在正前方某个位置。
- 2–5 秒：她的目光缓慢向一侧移动，又回到中间，嘴角轻轻上扬，右手握着手机停在胸前。
- 5–8 秒：她微微点了一下头，左手抬到半空像是在确认某个选择，眼神在两个位置之间短暂来回比较。
- 8–10 秒：她保持轻松的微笑，身体仍向前倾一点，视线稳定停在其中一个位置，像是已经基本选定。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
