# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
45岁左右的女性，齐肩深棕色短发，穿米色针织开衫和浅灰色衬衫，肩上背一个棕色皮质单肩包，手腕戴银色手表
背景：午休后路过办公楼便利区、想买一瓶饮品但担心不够划算的中年女职员

# State
- 购买阶段：desire
- Belief：她认为眼前的饮品很适合缓解下午的疲惫，但又觉得价格或必要性需要再斟酌。
- Desire：她明显想买下这件商品来提神解渴，同时希望自己的选择不要冲动浪费。
- Intention：她倾向于再确认几秒后做决定，手已经准备抬起选择但仍可能放弃。

# Behavior
- 可见行为：她的视线在前方不同位置来回移动，右手几次抬到胸前又停住，指尖微微蜷起，随后又慢慢放下，表现出明显犹豫和反复权衡。
- 表情：眉头轻微皱起，嘴唇轻抿，眼神专注但不确定，表情克制而纠结。
- 姿态：身体正面朝向镜头，上半身略微前倾，肩膀有些收紧，一只手扶着包带，另一只手在胸前犹豫地抬起又放下。

# Timeline
- 0–2 秒：这位45岁左右、穿米色针织开衫的女性站到镜头前，正面看向前方，眉头轻轻皱起，右手扶着棕色单肩包带。
- 2–5 秒：她的目光在前方左右轻微移动，像是在比较不同选择，嘴唇抿紧，右手慢慢抬到胸前却没有继续动作。
- 5–8 秒：她上半身稍微前倾，右手指尖悬停片刻后又缩回，表情更显犹豫，像是在衡量是否真的值得购买。
- 8–10 秒：她低低吸了一口气，右手再次抬起一点又放下，仍正面望着前方，停在一个尚未下定决心的购买前瞬间。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
