# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
20岁左右的男性，短黑发，穿浅灰色连帽卫衣和深色牛仔裤，背着一个黑色双肩包，手腕上戴着运动手环
背景：课间路过校园便利区的大学生，想买一瓶饮料但正在权衡是否值得花这笔钱。

# State
- 购买阶段：desire
- Belief：他认为眼前的商品很符合自己现在想喝点东西的需求，但价格可能略高，需要再确认是否划算。
- Desire：他想买到一款解渴又不太贵的饮品，同时避免冲动消费。
- Intention：他倾向于继续停在设备前比较和思考，暂时不进行操作。

# Behavior
- 可见行为：他始终面向镜头，视线在设备前方不同位置之间小幅移动，双手保持在胸前或腹前，手指轻微摩挲，偶尔停顿像是在反复衡量。
- 表情：表情克制而专注，眉头轻微收紧，嘴唇略微抿住，眼神表现出兴趣和思考中的犹豫。
- 姿态：身体微微前倾，肩膀放松但不完全松懈，双脚站定，重心偶尔轻微左右转移，呈现中等程度的犹豫感。

# Timeline
- 0–2 秒：20岁左右的男性学生走近后停在设备前，正面朝向镜头，眼睛迅速扫过前方，露出感兴趣的神情。
- 2–5 秒：他身体微微前倾，视线在前方来回移动，眉头轻轻皱起，像是在比较不同选择。
- 5–8 秒：他把双手放在腹前，手指轻微摩挲，嘴唇抿了一下，停顿片刻，像是在考虑价格是否合适。
- 8–10 秒：他仍然没有操作，只是保持正面站姿，目光再次短暂扫视前方，表情在兴趣和犹豫之间停留。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
