# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
20岁左右的男性，短黑发，穿浅灰色连帽卫衣和深色休闲裤，双肩背一个黑色书包，手里拿着一部手机
背景：课间路过校园便利区、被智能售货设备外观短暂吸引的男大学生

# State
- 购买阶段：attention
- Belief：他觉得前方设备的外观有点新奇，但还没有形成明确的购买判断。
- Desire：他只是想快速确认这是什么设备，满足一下路过时的好奇心。
- Intention：他倾向于边走边看一眼，然后继续前往原本的目的地。

# Behavior
- 可见行为：他从画面前方缓步经过，视线短暂抬向镜头附近，快速上下打量，手机握在一只手里，另一只手自然抓着书包肩带，没有停下操作。
- 表情：表情放松，眉毛轻微上扬，嘴角自然，带有一点短暂好奇但不明显犹豫。
- 姿态：身体保持正面朝向镜头，步伐连续但略微放慢，肩膀自然，背包贴在双肩，双手始终在画面内。

# Timeline
- 0–2 秒：20岁左右的男大学生正面走入画面，背着黑色书包，手里拿着手机，视线原本略微向前下方。
- 2–5 秒：他抬眼看向镜头附近，眉毛轻轻上扬，脚步没有停住，只是比刚进入画面时稍微放慢。
- 5–8 秒：他快速上下打量一眼，另一只手顺势抓紧书包肩带，脸上保持轻微好奇的放松表情。
- 8–10 秒：他收回视线，继续正面向前走过画面区域，动作自然，没有伸手操作或停留。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
