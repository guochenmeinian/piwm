# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
20岁左右的男性，短黑发，穿浅灰色连帽卫衣和深色休闲裤，肩上背一个黑色双肩包，手里拿着手机。
背景：课间路过校园便利区、被智能售货机内容吸引的男大学生。

# State
- 购买阶段：interest
- Belief：他认为眼前的选择里可能有适合自己课间补充能量的饮品或零食。
- Desire：他想快速了解有哪些看起来合适的选项，并挑出一个感兴趣的目标。
- Intention：他倾向继续查看不同区域，很快做出选择。

# Behavior
- 可见行为：他停在设备前，目光积极地在不同区域来回扫视，偶尔轻轻点头，双手自然放在画面内，一只手握着手机，动作轻松而连贯。
- 表情：表情放松，嘴角微微上扬，眼神明亮专注，明显带有兴趣但没有明显纠结。
- 姿态：身体略微前倾，肩膀放松，双脚站稳，头部随视线小幅转动，整体姿态自然主动。

# Timeline
- 0–2 秒：这名20岁左右的男大学生走近并停在镜头前，身体自然前倾，眼神迅速落到前方。
- 2–5 秒：他微微抬头又低头，目光轻快地在不同区域扫视，握着手机的手保持在胸前附近。
- 5–8 秒：他嘴角略微上扬，轻轻点了一下头，另一只手放松地抬起一点，像是在确认某个感兴趣的选择。
- 8–10 秒：他继续保持前倾姿态，视线再次横向扫过，表情积极而放松，准备进一步选择。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
