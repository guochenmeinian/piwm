# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
28岁左右的女性，齐肩黑色直发，穿浅灰色针织开衫和白色圆领上衣，背一个米色帆布托特包，手里拿着一部手机。
背景：午休间隙路过便利区、想买饮品但担心不够健康的年轻上班族。

# State
- 购买阶段：interest
- Belief：她认为这里可能有适合午后提神的选择，但不确定是否符合自己的健康需求。
- Desire：她想快速找到一款低负担、不会太甜的饮品。
- Intention：她倾向于继续观察和比较片刻，再决定是否购买。

# Behavior
- 可见行为：她站在设备前保持正面朝向，视线在前方不同区域频繁转移，偶尔低头看手机又抬头观察，手指轻轻摩挲手机边缘，没有立即操作。
- 表情：眉头微微收紧，嘴唇轻抿，表情专注但带有明显犹豫。
- 姿态：身体重心略微向后，肩膀有些收紧，一只脚轻轻后撤，双手始终在胸前或腰前画面内。

# Timeline
- 0–2 秒：这名28岁左右的女性正面站定，抬眼看向前方，右手握着手机，左手轻扶托特包肩带。
- 2–5 秒：她的视线从左侧移到右侧又回到中间，眉头微皱，手指无意识地摩挲手机边缘。
- 5–8 秒：她稍稍向后移了半步，低头短暂看了一眼手机，再抬头继续观察，嘴唇轻轻抿起。
- 8–10 秒：她停在原地没有动作选择，双手仍在画面内，身体保持微微后倾，脸上维持犹豫而感兴趣的神情。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
