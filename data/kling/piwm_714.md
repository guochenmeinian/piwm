# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
28岁左右的男性，短黑发，穿浅灰色针织开衫和白色T恤，斜挎一个深棕色通勤包，手里拿着手机
背景：午休结束前路过办公楼便利区的年轻上班族，想快速买一瓶饮品但还在两个选择之间比较。

# State
- 购买阶段：desire
- Belief：他认为眼前两个选择都适合现在喝，一个更解渴，另一个看起来更有新鲜感。
- Desire：他想买一瓶让自己下午更舒服的饮品，但希望在口味和实用性之间选得更稳妥。
- Intention：他倾向于再看一眼两个方向后做出选择，准备把手伸向其中一个选项。

# Behavior
- 可见行为：他的目光在左侧和右侧两个方向之间来回切换，偶尔轻轻点头又停住，握着手机的手指轻微摩挲，另一只手的食指小幅抬起又放下。
- 表情：表情专注但带有中等程度犹豫，眉头轻微收紧，嘴唇短暂抿起，眼神在比较中显得认真克制。
- 姿态：身体正面朝向镜头，肩膀略向前倾，双手保持在胸前到腰部范围内，重心轻微左右转换但动作幅度不大。

# Timeline
- 0–2 秒：28岁左右的男性正面站定，视线先停在左侧方向，眉头微收，握着手机的手指轻轻摩挲。
- 2–5 秒：他把目光移向右侧方向，头部轻微跟随转动，像是在比较另一个选择，嘴唇短暂抿起。
- 5–8 秒：他轻轻点了一下头又马上收住，另一只手的食指小幅抬起后放下，视线再次在左侧和右侧之间切换。
- 8–10 秒：他身体略微前倾，目光停留在其中一个方向，表情仍有些犹豫但更接近决定，手指准备向前伸出。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
