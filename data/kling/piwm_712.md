# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
28岁左右的男性，短黑发，穿浅灰色衬衫和深色休闲西裤，左肩背一个黑色通勤包，手腕戴银色手表
背景：下班途中经过办公楼便利区、准备快速买一瓶饮料的年轻上班族

# State
- 购买阶段：desire
- Belief：他已经判断自己想要的那款饮料就在面前的固定位置，且符合当前解渴和提神的需求。
- Desire：他想尽快确认选择并买下那款饮料，减少下班路上的停留时间。
- Intention：他接下来倾向于直接伸手操作或确认购买，几乎不再继续比较其他选项。

# Behavior
- 可见行为：他的目光持续锁定在画面下方偏中央的固定位置，双手保持在胸前到腰部范围内，手指轻微收拢又放松，只有短暂的小幅确认动作。
- 表情：表情平静专注，眉眼放松但视线稳定，嘴唇自然闭合，几乎没有明显犹豫神色。
- 姿态：身体微微前倾，肩膀保持稳定，头部略向前探，站姿端正且重心靠近前方。

# Timeline
- 0–2 秒：28岁左右的男性上班族正面站定，身体微微前倾，视线迅速落在画面下方偏中央的固定位置。
- 2–5 秒：他保持平静专注的表情，眼神几乎不移动，双手在画面内自然抬起到腰腹前方，手指轻微收拢。
- 5–8 秒：他略微点头般确认了一下，肩膀和身体继续向前保持轻微倾斜，视线仍牢牢锁定同一位置。
- 8–10 秒：他的右手开始向前小幅伸出，表情仍然稳定专注，呈现即将做出购买决定的状态。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
