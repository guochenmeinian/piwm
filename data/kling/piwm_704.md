# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
25岁左右的女性，齐肩黑色直发，穿浅灰色针织开衫和白色内搭，背着米色帆布单肩包，手里拿着手机
背景：下班途中经过便利区、被智能售货机吸引而停下脚步的年轻女性上班族

# State
- 购买阶段：attention
- Belief：她刚注意到前方设备似乎有值得查看的内容，但还不确定是否与自己当前需求相关。
- Desire：她想快速判断这里是否有适合下班路上购买的饮品或小食。
- Intention：她倾向先停下来观察几秒，再决定是否进一步靠近或操作。

# Behavior
- 可见行为：她从经过状态放慢并停住，视线被前方吸引，头部微微转向设备方向，手机握在胸前但没有操作，另一只手轻轻扶着包带。
- 表情：眉毛轻微上扬，眼神专注但带一点疑问，嘴唇自然闭合，表情克制而犹豫。
- 姿态：身体略微斜向一侧，尚未完全正面站定，肩膀有轻微转动，重心停在一只脚上，姿态表现出中等程度的迟疑。

# Timeline
- 0–2 秒：这位25岁左右的女性从画面前方经过时放慢脚步，眼神被前方吸引，头部轻轻转向设备方向。
- 2–5 秒：她停在原地，身体仍略微侧着，右手握着手机停在胸前，左手自然扶住米色帆布包带。
- 5–8 秒：她微微抬眉，视线在前方上下扫过，像是在确认内容是否值得继续查看，动作保持克制。
- 8–10 秒：她没有立即行动，只是稍稍调整站姿并继续注视前方，脸上保持轻微疑问和中等犹豫。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
