# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
28岁左右的女性，黑色中长发自然披肩，穿浅米色风衣和白色针织内搭，肩背一个棕色通勤托特包，手里拿着手机和门禁卡。
背景：下班路过办公楼便利区的年轻白领女性，想顺手看看是否有合适的饮品。

# State
- 购买阶段：interest
- Belief：她认为眼前的饮品选择可能适合下班后解渴或提神，但还需要简单比较一下。
- Desire：她想快速找到一款口味合适、喝起来轻松的饮品。
- Intention：她倾向继续平稳扫视并做一个简单判断，若看到合适选项就准备购买。

# Behavior
- 可见行为：她站在设备前方，目光平稳地从左到右扫视，偶尔短暂停留，手指轻轻握着手机，动作自然放松，没有明显反复犹豫。
- 表情：表情平和专注，嘴角自然放松，眼神稳定，带有轻微兴趣感。
- 姿态：身体正面朝向镜头，肩膀放松，重心稳定，双手保持在胸前到腰间的画面范围内，姿态轻松自然。

# Timeline
- 0–2 秒：28岁左右的女性走近并停在画面中央，正面朝向镜头，肩背棕色托特包，手里自然拿着手机和门禁卡。
- 2–5 秒：她的目光从画面左侧平稳扫到右侧，头部只做轻微转动，表情保持平和专注。
- 5–8 秒：她在某个方向短暂停留视线，手指轻轻调整手机位置，身体微微前倾但整体仍然放松。
- 8–10 秒：她收回轻微前倾的姿态，目光再次平稳扫过前方，露出低程度犹豫后的自然兴趣神情。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
