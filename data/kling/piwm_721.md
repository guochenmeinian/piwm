# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
20岁左右的男性，短黑发，穿浅灰色连帽卫衣和深色休闲裤，双肩背一个黑色书包，手里拿着一部手机。
背景：晚自习前路过校园便利区、准备快速购买饮品的大学生。

# State
- 购买阶段：action
- Belief：他认为自己已经基本选定要买的东西，只需要完成最后确认即可。
- Desire：他想在不耽误晚自习的情况下尽快完成购买。
- Intention：他倾向于短暂停顿确认后继续完成操作。

# Behavior
- 可见行为：他站在镜头前，手机握在一只手中，另一只手抬起准备操作，动作在半空中短暂停住，目光在下方操作区和旁侧位置之间快速移动后继续伸手。
- 表情：表情克制而专注，眉头轻微收紧，嘴唇自然闭合，带有中等程度的犹豫感。
- 姿态：身体正面朝向镜头，肩膀略微前倾，双手保持在画面内，重心稳定但上身有轻微停顿感。

# Timeline
- 0–2 秒：20岁左右的男性学生正面站定，低头看向下方操作区，右手握着手机，左手缓慢抬起。
- 2–5 秒：他准备继续操作时手指在半空中停了一下，眉头微微收紧，目光短暂移向旁侧位置。
- 5–8 秒：他又看回下方操作区，左手小幅调整位置，手指犹豫地悬停片刻后向前移动。
- 8–10 秒：他轻轻点下并保持专注表情，身体仍正面朝向镜头，动作显示已进入最后确认。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
