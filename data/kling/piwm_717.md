# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
28岁左右的男性，短黑发，穿浅灰色衬衫和深色休闲西裤，胸前挂着工牌，肩上背一个黑色通勤双肩包。
背景：下班后路过办公楼便利区、想买点东西补充能量的年轻上班族。

# State
- 购买阶段：desire
- Belief：他认为眼前的选择能缓解下班后的疲惫，但又担心不够健康或不是最划算的选择。
- Desire：他想尽快选到一个既能满足当下需求又不会让自己后悔的商品。
- Intention：他倾向于购买，但还想再权衡几秒钟后再做决定。

# Behavior
- 可见行为：他反复在前方不同位置之间移动视线，右手微微抬起像要操作，停顿后又放下，左手轻轻捏住背包肩带，动作迟疑而克制。
- 表情：眉头轻皱，嘴唇微抿，眼神在专注和犹豫之间来回变化，表情明显纠结但不夸张。
- 姿态：身体正面朝向镜头，上半身略微前倾，肩膀有些紧绷，重心在双脚之间小幅转移，呈现反复权衡的状态。

# Timeline
- 0–2 秒：这名28岁左右、穿浅灰色衬衫的男性上班族走到画面中央并停下，正面看向前方，双手都保持在画面内。
- 2–5 秒：他微微前倾，视线从左到右缓慢扫过，左手捏住黑色双肩包肩带，眉头逐渐皱起。
- 5–8 秒：他的右手抬到胸前高度，像是准备做出选择，却在半空停住，嘴唇抿紧，随后又把手慢慢放下。
- 8–10 秒：他轻轻换了一下重心，再次看向前方某处，表情仍然纠结，保持犹豫但接近决定的姿态。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
