# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
28岁左右的男性，短黑发，穿浅灰色衬衫和深色休闲西裤，肩上背一个黑色通勤双肩包，一只手腕戴银色手表。
背景：下班途中路过便利区、想快速买一瓶饮料的年轻上班族。

# State
- 购买阶段：action
- Belief：他认为自己已经基本选定了想买的饮品，但仍想确认没有选错。
- Desire：他想尽快完成购买，同时避免因为匆忙按错选项。
- Intention：他倾向于伸手进行操作，但会在确认后再继续完成选择。

# Behavior
- 可见行为：他站在镜头前，目光在正前方和略下方之间短暂切换，右手伸向画面下方准备操作，中途停顿一下后又继续前伸。
- 表情：表情专注而略带犹豫，眉头轻微收紧，嘴唇自然闭合，没有夸张情绪。
- 姿态：身体正面朝向镜头，上半身微微前倾，左手靠近背包肩带，右手抬起并保持在画面内，整体姿态克制稳定。

# Timeline
- 0–2 秒：这名28岁左右的男性上班族正面站定，目光先落在正前方，右手从身体一侧慢慢抬起。
- 2–5 秒：他微微前倾，右手伸向画面下方准备操作，眼神保持专注，眉头轻轻皱起。
- 5–8 秒：他的右手在半空中短暂停住，动作中断了一下，眼睛又快速向正前方看了一眼，像是在再次确认选择。
- 8–10 秒：确认后他放松一点，右手继续向前下方伸出完成操作，身体仍保持正面朝向镜头。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
