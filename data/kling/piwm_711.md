# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
20岁左右的男性，短黑发，穿浅灰色连帽卫衣和深色休闲裤，双肩背一个黑色书包，手里拿着手机
背景：课间经过校园便利区的大学生，想快速选一份饮品但还没拿定主意。

# State
- 购买阶段：interest
- Belief：他认为面前某个选择可能适合课间补充能量，但还不确定是否值得立刻购买。
- Desire：他想找到一款既合适又不会后悔的饮品。
- Intention：他倾向继续盯着感兴趣的方向观察比较，暂时不马上下单。

# Behavior
- 可见行为：他反复看向画面一侧的固定方向，手指轻轻摩挲手机边缘，短暂停顿后后退一小步，仍保持视线关注同一方向。
- 表情：眉头微微收紧，嘴唇轻抿，眼神专注但带有明显犹豫。
- 姿态：身体正面朝向镜头，上半身略微前倾后又收回，重心轻微后移，双手保持在胸前到腰部之间的画面内。

# Timeline
- 0–2 秒：20岁左右的男性学生正面站在镜头前，目光先扫过前方后停在画面右侧，双手握着手机放在胸前。
- 2–5 秒：他微微眯眼看向同一方向，拇指无意识摩挲手机边缘，眉头逐渐收紧，表现出明显兴趣。
- 5–8 秒：他轻轻吸气，身体重心向后移，右脚后退一小步，同时视线仍没有离开感兴趣的方向。
- 8–10 秒：他停在稍远的位置，嘴唇轻抿，双手仍握着手机，犹豫地继续观察而没有进一步操作。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
