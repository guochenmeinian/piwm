# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
20岁左右的男性，短黑发，穿浅灰色连帽卫衣和深色牛仔裤，肩背一个深蓝色双肩包
背景：课间路过校园便利区、已经决定购买饮料的大学生

# State
- 购买阶段：action
- Belief：他认为自己已经选好了合适的饮料，购买步骤很简单且不需要再比较。
- Desire：他想快速完成购买，在下一节课前拿到饮料。
- Intention：他准备立即伸手操作并确认购买。

# Behavior
- 可见行为：他始终正面看向前方，视线稳定，动作直接，右手从胸前自然抬起并向画面下方前伸，左手轻扶背包肩带，全程没有停顿、回看或反复比较的动作。
- 表情：表情轻松而坚定，嘴角微微放松，眼神专注，没有皱眉或犹豫神态。
- 姿态：上半身明显前倾，肩膀放松，双脚站稳，右臂伸出准备操作，姿态主动且确定。

# Timeline
- 0–2 秒：这名20岁左右的男学生正面站定，目光平稳看向前方，脸上带着轻松坚定的神情。
- 2–5 秒：他上半身自然前倾，左手扶住深蓝色双肩包肩带，右手从身体前方抬起。
- 5–8 秒：他保持专注视线，右臂继续向画面下方前伸，动作流畅直接，没有任何迟疑。
- 8–10 秒：他右手停在准备操作的位置，身体仍微微前倾，表情放松坚定，显然已经准备确认购买。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
