# Task
生成一段 10 秒、单镜头、固定机位、写实自然风格的视频，模拟智能售货机前置摄像头记录到的顾客行为片段。

# Camera
镜头来自智能售货机正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。你能不受视野遮挡地看到顾客周身，画面中**禁止出现**机器本体、商品、货架、价格标签或任何商品。

# Character
45岁左右的男性，短黑发略带灰白，穿浅灰色夹克和深色休闲裤，左手拿着手机，右手拎着一个黑色公文包。
背景：午休后路过便利区的中年办公室职员，只是顺路经过并没有明确购买计划。

# State
- 购买阶段：attention
- Belief：他隐约注意到前方有可购买饮品或零食的零售设备，但不确定是否有自己需要的东西。
- Desire：他只想快速判断这里是否值得停留，避免耽误返回办公室的时间。
- Intention：他倾向于短暂扫视后直接离开，除非立刻发现非常合适的选择。

# Behavior
- 可见行为：他在镜头前短暂停步，视线从上到下快速扫过前方区域，随后目光游移，左手手机停在胸前，右手公文包轻微晃动，没有进一步靠近或操作的动作。
- 表情：眉头轻微皱起，嘴角放平，眼神短暂聚焦后变得犹疑，整体表情略显困惑且兴趣不高。
- 姿态：身体正面朝向镜头但重心偏向一侧，肩膀微微转开，右脚已经向外侧迈出半步，呈现准备转身离开的姿态。

# Timeline
- 0–2 秒：45岁左右的男性从画面正前方进入并放慢脚步，抬眼快速看向前方区域，左手握着手机停在胸前。
- 2–5 秒：他短暂停住，视线上下扫动一次，眉头轻轻皱起，右手拎着的黑色公文包在腿侧小幅摆动。
- 5–8 秒：他没有继续靠近，目光偏向一侧又回到前方，嘴唇微抿，身体重心逐渐移到准备离开的脚上。
- 8–10 秒：他肩膀微微转开，右脚向外侧迈出半步，仍保持面部可见，带着困惑和低兴趣的表情准备离开。

# Constraints
- 顾客外观必须与 Character 描述完全一致，全程不变
- 禁止出现：售货机本体、机器边框、商品、货架
- 无夸张表情、无夸张动作、无表演感，无其他人介入交互
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
