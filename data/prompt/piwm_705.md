# Task
请生成一段 10 秒、单镜头、固定机位、无剪辑、写实自然风格的视频。

# Camera
镜头来自智能售货机或智能冰箱正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。画面中不需要出现机器本体、屏幕内容、商品、货架、价格标签或任何商品信息。

# Scene
室内零售环境，明亮自然的展示照明，背景干净、安静、日常。无人与顾客互动，无对白，无字幕，无杂音。

# Behavior State
- 当前购买阶段：interest
- 购买兴趣强度：中等
- 犹豫程度：中等
- 主要可见行为：顾客站在镜头前方中央，目光在左侧和右侧之间缓慢来回移动，偶尔轻微停顿，双手自然垂放或一只手轻触衣角，没有伸手操作。
- 表情状态：表情淡漠、疲惫，眼神略显空，眉眼放松但缺少兴奋感，像是工作后精力不足。
- 身体姿态：身体正面朝向镜头，肩膀略微下沉，上半身轻微前倾，站姿稳定但显得有些松散。

# Timeline
- 0–2 秒：顾客走近后停在画面中央，抬眼看向左侧区域，脸上保持淡漠疲惫的表情。
- 2–5 秒：顾客的目光从左侧缓慢移到右侧，头部只做小幅转动，肩膀略微下沉。
- 5–8 秒：顾客在右侧短暂停留后又看回左侧，嘴唇轻轻抿住，双手仍未抬起操作。
- 8–10 秒：顾客再次在左右之间扫视，身体保持轻微前倾，仍然没有决定接下来要看哪一边。

# Constraints
- 顾客必须始终正面可见
- 面部无遮挡，五官清晰
- 不能出现任何商品、货架、价格标签、机器本体或屏幕 UI
- 无其他人介入交互
- 无夸张表情、无夸张动作、无表演感
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
