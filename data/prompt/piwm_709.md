# Task
请生成一段 10 秒、单镜头、固定机位、无剪辑、写实自然风格的视频。

# Camera
镜头来自智能售货机或智能冰箱正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。画面中不需要出现机器本体、屏幕内容、商品、货架、价格标签或任何商品信息。

# Scene
室内零售环境，明亮自然的展示照明，背景干净、安静、日常。无人与顾客互动，无对白，无字幕，无杂音。

# Behavior State
- 当前购买阶段：action
- 购买兴趣强度：高
- 犹豫程度：中等
- 主要可见行为：顾客站在画面中央，双手抬在胸前靠近操作区，先开始进行操作，中途手指停住，目光从下方操作区短暂移回确认位置，停顿片刻后再次低头继续操作。
- 表情状态：表情克制自然，眉头轻微收紧，嘴唇短暂抿起，呈现略显不确定但仍决定继续的状态。
- 身体姿态：上半身略微前倾，肩膀放松但注意力集中，双肘贴近身体，停顿时身体保持稳定，随后轻微向前靠近继续操作。

# Timeline
- 0–2 秒：顾客正面站在画面中央，目光向下落在操作区，右手抬起并开始进行一次明确的操作动作。
- 2–5 秒：她的手指在半途中停住，动作变慢，眼神从操作区移回较高的确认位置，像是在重新核对当前选择。
- 5–8 秒：她保持短暂停顿，眉头轻轻收紧，嘴唇微抿，左手略微收回，右手悬停在原处没有立刻继续。
- 8–10 秒：她表情恢复平稳，目光重新回到操作区，右手继续完成操作，上半身轻微前倾但动作仍然克制自然。

# Constraints
- 顾客必须始终正面可见
- 面部无遮挡，五官清晰
- 不能出现任何商品、货架、价格标签、机器本体或屏幕 UI
- 无其他人介入交互
- 无夸张表情、无夸张动作、无表演感
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
