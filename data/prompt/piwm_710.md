# Task
请生成一段 10 秒、单镜头、固定机位、无剪辑、写实自然风格的视频。

# Camera
镜头来自智能售货机或智能冰箱正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。画面中不需要出现机器本体、屏幕内容、商品、货架、价格标签或任何商品信息。

# Scene
室内零售环境，明亮自然的展示照明，背景干净、安静、日常。无人与顾客互动，无对白，无字幕，无杂音。

# Behavior State
- 当前购买阶段：action
- 购买兴趣强度：高
- 犹豫程度：中等
- 主要可见行为：顾客站在画面中央，目光多次在下方操作区方向和旁侧位置之间短暂移动，一只手抬起接近操作区域又略微停住，手指有轻微收回和再伸出的动作。
- 表情状态：表情专注但带一点迟疑，眉头轻微收紧，嘴唇自然闭合，眼神在确认与犹豫之间切换。
- 身体姿态：上半身略微前倾，肩膀放松，双手位于胸前到腰部之间，一只手准备操作但动作不够果断，整体保持正面朝向镜头。

# Timeline
- 0–2 秒：顾客正面站定在画面中央，微微前倾，目光先落向下方操作区，右手从身侧抬起到胸前。
- 2–5 秒：他保持专注表情，眼睛短暂移向旁侧位置像是在再次核对，抬起的手停在半空，手指轻轻弯曲。
- 5–8 秒：顾客的视线回到下方操作区，右手向前伸出一点又稍微收回，眉头轻微收紧，表现出最后确认时的中等犹豫。
- 8–10 秒：他短暂停顿后再次看向操作区，手指缓慢靠近确认位置，身体仍保持前倾，像是即将完成购买确认。

# Constraints
- 顾客必须始终正面可见
- 面部无遮挡，五官清晰
- 不能出现任何商品、货架、价格标签、机器本体或屏幕 UI
- 无其他人介入交互
- 无夸张表情、无夸张动作、无表演感
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
