# Task
请生成一段 10 秒、单镜头、固定机位、无剪辑、写实自然风格的视频。

# Camera
镜头来自智能售货机或智能冰箱正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。画面中不需要出现机器本体、屏幕内容、商品、货架、价格标签或任何商品信息。

# Scene
室内零售环境，明亮自然的展示照明，背景干净、安静、日常。无人与顾客互动，无对白，无字幕，无杂音。

# Behavior State
- 当前购买阶段：desire
- 购买兴趣强度：高
- 犹豫程度：低
- 主要可见行为：她先短暂扫视前方，随后视线固定在一个位置，右手从口袋里拿出手机并将手机握在胸前，动作自然且明确。
- 表情状态：表情由轻微思考和确认感逐渐转为放松、确定，嘴角微收，眼神稳定。
- 身体姿态：身体正对镜头，上半身略微前倾，肩膀放松，双手动作集中在取出手机和准备操作上。

# Timeline
- 0–2 秒：她站在画面中央，正面朝向镜头，目光在前方轻轻扫过，脸上带着短暂思考的神情。
- 2–5 秒：她的视线逐渐停在固定位置，眉头轻微舒展，像是已经确认了想要的选择。
- 5–8 秒：她低头幅度很小地看向衣服口袋方向，右手伸进口袋并自然取出手机，表情变得更确定。
- 8–10 秒：她把手机握在胸前并看回前方固定位置，身体微微前倾，呈现准备购买的状态。

# Constraints
- 顾客必须始终正面可见
- 面部无遮挡，五官清晰
- 不能出现任何商品、货架、价格标签、机器本体或屏幕 UI
- 无其他人介入交互
- 无夸张表情、无夸张动作、无表演感
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
