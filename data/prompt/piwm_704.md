# Task
请生成一段 10 秒、单镜头、固定机位、无剪辑、写实自然风格的视频。

# Camera
镜头来自智能售货机或智能冰箱正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。画面中不需要出现机器本体、屏幕内容、商品、货架、价格标签或任何商品信息。

# Scene
室内零售环境，明亮自然的展示照明，背景干净、安静、日常。无人与顾客互动，无对白，无字幕，无杂音。

# Behavior State
- 当前购买阶段：interest
- 购买兴趣强度：高
- 犹豫程度：低
- 主要可见行为：他停在镜头前，目光积极地在不同区域之间来回扫视，偶尔轻微点头，双手自然停在身前或背包肩带附近，没有明显退缩或反复纠结动作。
- 表情状态：表情放松但明显带有兴趣，眼神专注，眉毛微微上扬，嘴角有轻微上扬的倾向。
- 身体姿态：上半身略微前倾，正面朝向镜头，站姿稳定，肩膀放松，身体重心靠近设备。

# Timeline
- 0–2 秒：男大学生从画面中央前方停下，正面朝向镜头，上半身自然前倾，目光迅速落到前方某一区域。
- 2–5 秒：他的眼睛积极地从左侧扫到右侧，再略微向下看，眉毛轻抬，表现出对可选内容的兴趣。
- 5–8 秒：他保持靠近设备的姿态，轻轻调整一侧背包肩带，目光在几个区域之间快速比较，表情专注而轻松。
- 8–10 秒：他微微点头，视线停在前方某个位置，身体仍然前倾，像是准备继续下一步操作。

# Constraints
- 顾客必须始终正面可见
- 面部无遮挡，五官清晰
- 不能出现任何商品、货架、价格标签、机器本体或屏幕 UI
- 无其他人介入交互
- 无夸张表情、无夸张动作、无表演感
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
