# Task
请生成一段 10 秒、单镜头、固定机位、无剪辑、写实自然风格的视频。

# Camera
镜头来自智能售货机或智能冰箱正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。画面中不需要出现机器本体、屏幕内容、商品、货架、价格标签或任何商品信息。

# Scene
室内零售环境，明亮自然的展示照明，背景干净、安静、日常。无人与顾客互动，无对白，无字幕，无杂音。

# Behavior State
- 当前购买阶段：action
- 购买兴趣强度：高，已转化为明确购买行动
- 犹豫程度：极低，几乎没有犹豫
- 主要可见行为：目光稳定看向前下方操作区域，右手直接抬起并向前完成一次确认动作，动作短促干净，没有反复查看、停顿或回头确认。
- 表情状态：中性放松，眉眼平稳，嘴角自然，表情没有紧张、迟疑或纠结感。
- 身体姿态：身体正面朝向镜头，站姿稳定，肩膀放松，上半身略微前倾以完成操作，随后恢复自然直立。

# Timeline
- 0–2 秒：顾客位于画面中央，正面站定，目光直接落在前下方操作区域，表情中性放松。
- 2–5 秒：他几乎不做停顿，右手从身体一侧抬起，手指自然伸出，动作明确地向前靠近进行确认。
- 5–8 秒：右手完成一次简短直接的确认动作后立即收回，视线保持稳定，没有再次扫视或回头检查。
- 8–10 秒：他上半身恢复自然直立，神情平静，身体轻微转向准备离开，整个过程没有任何犹豫迹象。

# Constraints
- 顾客必须始终正面可见
- 面部无遮挡，五官清晰
- 不能出现任何商品、货架、价格标签、机器本体或屏幕 UI
- 无其他人介入交互
- 无夸张表情、无夸张动作、无表演感
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
