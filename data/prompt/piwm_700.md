# Task
生成一段 10 秒、单镜头、固定机位、无剪辑、无切换视角、写实自然风格的视频。
视频内容是：一名顾客站在智能售货机前，通过外显行为、表情、目光和姿态体现其当前购买状态。

# Camera
镜头来自一台智能售货机正面上方中央的前置摄像头。
摄像头朝外拍摄顾客，为第一人称视角，轻微俯视，固定机位，画面稳定，不晃动，不变焦。

画面要求：
- 摄像头视野完全不受遮挡
- 顾客始终位于画面中央附近
- 顾客正面朝向镜头
- 清晰可见顾客的面部、目光方向、细微表情、上半身和双手
- 面部无遮挡，双手始终在画面内
- 画面中不要出现机器本体、屏幕内容、商品、货架、价格标签或任何商品信息
- 画面里只呈现顾客和前方环境

# Scene
室内零售环境，明亮但自然的展示照明。
背景干净、安静、日常。
无其他顾客，无店员，无人与顾客互动。
无对白，无字幕，无配音，无杂音。

# Customer
一名 25–35 岁的普通亚洲女性顾客，穿着简洁自然的日常休闲服饰，外表生活化，气质自然克制，像真实逛店的普通消费者。

# Behavior State
- 当前购买阶段：interest
- 购买兴趣强度：较高，愿意停留继续观察，对商品有明显兴趣。
- 犹豫程度：中等偏高，正在比较和确认信息，但尚未做出购买决定。
- 主要可见行为：顾客走近后在机器前停留，身体轻微前倾，视线先稳定看向前方，再短暂下移，并在前方不同位置之间自然切换数次，持续表现出浏览、比较和确认信息的状态。
- 表情状态：表情自然克制，带有轻微思考、权衡和犹豫，没有夸张惊讶或明显情绪波动。
- 身体姿态：正面朝向机器，站姿稳定，停留在原地，身体轻微前倾，双手自然放置并始终保持在画面内。

# Behavior Direction
顾客的行为应自然体现其当前购买心理状态。
动作和表情要克制、生活化，像真实顾客在零售设备前浏览时的微行为。
不要让顾客像演员一样刻意表演。
不要通过商品、价格标签、屏幕 UI 或机器本体来表达状态，只通过顾客的目光、停留、表情、身体姿态和轻微动作来体现。

# Timeline
- 0–2 秒：顾客从前方走近，进入镜头中心，脚步逐渐放慢，在机器前停下，并将注意力集中到机器前方开始认真观察。
- 2–5 秒：顾客继续停留，身体轻微前倾，目光稳定看向前方，随后短暂下移视线，再回到前方，像是在浏览和确认不同信息。
- 5–8 秒：顾客头部和目光在前方不同位置之间自然切换数次，节奏缓慢真实，表现出比较、思考和犹豫。
- 8–10 秒：顾客继续停留在原地，再次短暂下移视线，然后回到前方，保持思考和犹豫状态，没有支付、没有扫码、没有离开。

# Constraints
- 只有一名顾客
- 顾客必须始终正面可见
- 面部无遮挡，五官清晰
- 双手始终在画面内
- 摄像头视野完全不受遮挡
- 不能出现任何商品、货架、价格标签、机器本体或屏幕 UI
- 无其他人进入画面
- 无人与顾客互动
- 无对白、无配音、无字幕、无杂音
- 无夸张表情、无夸张动作、无表演感
- 无扫码、无支付、无购买完成
- 无转身离开
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
