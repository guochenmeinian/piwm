# Task
请生成一段 10 秒、单镜头、固定机位、无剪辑、写实自然风格的视频。

# Camera
镜头来自智能售货机或智能冰箱正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。画面中不需要出现机器本体、屏幕内容、商品、货架、价格标签或任何商品信息。

# Scene
室内零售环境，明亮自然的展示照明，背景干净、安静、日常。无人与顾客互动，无对白，无字幕，无杂音。

# Behavior State
- 当前购买阶段：action
- 购买兴趣强度：高，购买意图明确，注意力集中在完成操作而非继续浏览。
- 犹豫程度：极低，几乎没有停顿或反复确认。
- 主要可见行为：她的目光稳定落在下方操作区，双手动作连贯自然，短暂抬手点击或确认后迅速回到准备取物的姿态，整体表现像熟悉流程的常客。
- 表情状态：表情平静自然，眉眼放松，嘴唇轻闭，偶尔有轻微专注感但没有紧张或犹豫。
- 身体姿态：身体微微前倾，肩膀放松，上半身正对镜头，双手位于胸前到腰腹之间的操作范围内，姿态稳定利落。

# Timeline
- 0–2 秒：年轻女性站在画面中央附近，身体微前倾，平静地将目光落向下方操作区，双手自然抬起准备操作。
- 2–5 秒：她用一只手做出流畅的点击或确认动作，另一只手轻扶在身前，面部保持专注且没有明显犹豫。
- 5–8 秒：她短暂停顿等待反馈，眼神仍固定在操作区，随后手指再次轻快地完成一个确认动作，动作节奏熟练自然。
- 8–10 秒：她的双手放松下移，身体仍略向前倾，表情平静，像是已经完成购买步骤并准备取走商品。

# Constraints
- 顾客必须始终正面可见
- 面部无遮挡，五官清晰
- 不能出现任何商品、货架、价格标签、机器本体或屏幕 UI
- 无其他人介入交互
- 无夸张表情、无夸张动作、无表演感
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
