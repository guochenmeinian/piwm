# Task
请生成一段 10 秒、单镜头、固定机位、无剪辑、写实自然风格的视频。

# Camera
镜头来自智能售货机或智能冰箱正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。画面中不需要出现机器本体、屏幕内容、商品、货架、价格标签或任何商品信息。

# Scene
室内零售环境，明亮自然的展示照明，背景干净、安静、日常。无人与顾客互动，无对白，无字幕，无杂音。

# Behavior State
- 当前购买阶段：attention
- 购买兴趣强度：低
- 犹豫程度：高
- 主要可见行为：她站在设备前侧附近，目光多次从镜头方向滑开，偶尔低头看向手边或侧方，双手轻轻交叠又松开，没有接近操作的动作。
- 表情状态：表情平静但略显走神，眉眼放松，嘴角中性，偶尔轻微抿唇，缺少明确兴奋或购买期待。
- 身体姿态：身体正面朝向镜头但重心偏向一侧，肩膀自然下垂，站姿略有等待感，上半身偶尔细微转动。

# Timeline
- 0–2 秒：她站在画面中央附近，身体略偏向一侧，目光没有聚焦在镜头方向，像是刚停下来等人。
- 2–5 秒：她缓慢扫视前方，又短暂看向侧边，双手在身前轻轻交叠，表情平淡而有些分心。
- 5–8 秒：她微微抿唇，低头看了一眼手边位置后又抬眼随意打量，仍没有任何准备操作或购买的动作。
- 8–10 秒：她把重心换到另一只脚，目光再次飘向远处，继续安静等待，兴趣仍然很低且犹豫不决。

# Constraints
- 顾客必须始终正面可见
- 面部无遮挡，五官清晰
- 不能出现任何商品、货架、价格标签、机器本体或屏幕 UI
- 无其他人介入交互
- 无夸张表情、无夸张动作、无表演感
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
