# Task
请生成一段 10 秒、单镜头、固定机位、无剪辑、写实自然风格的视频。

# Camera
镜头来自智能售货机或智能冰箱正面上方的前置摄像头。第一人称视角，轻微俯视，固定机位，不晃动，不变焦。摄像头朝外拍摄顾客，视野完全不受遮挡。画面中清晰可见顾客的面部、目光方向、细微表情、上半身和双手。顾客始终位于画面中央附近，正面朝向镜头，面部无遮挡，双手始终在画面内。画面中不需要出现机器本体、屏幕内容、商品、货架、价格标签或任何商品信息。

# Scene
室内零售环境，明亮自然的展示照明，背景干净、安静、日常。无人与顾客互动，无对白，无字幕，无杂音。

# Behavior State
- 当前购买阶段：interest
- 购买兴趣强度：中等偏低，表现为愿意停留观察但没有强烈购买冲动
- 犹豫程度：低，几乎没有明显纠结，只是进行轻度比较
- 主要可见行为：顾客站在设备前，目光在不同方向之间平稳移动，偶尔轻微点头，双手自然放在身前或一手握着手机，没有触摸或取物动作。
- 表情状态：表情平静自然，眼神专注但不兴奋，嘴唇放松，偶尔露出轻微思考神情。
- 身体姿态：上半身正对镜头，身体略微前倾，肩膀放松，站姿稳定，动作幅度小且自然。

# Timeline
- 0–2 秒：顾客从画面中央稳定站定，抬眼看向前方，表情平静，像是刚被某个选项吸引而停下。
- 2–5 秒：她的目光缓慢从左侧移到右侧，头部轻微转动，双手保持自然放松，表现出一般程度的浏览兴趣。
- 5–8 秒：她稍稍前倾半步，眼睛短暂停在某个方向，眉头轻微收起又放松，像是在快速比较但没有明显犹豫。
- 8–10 秒：她轻微点头后恢复直立，视线仍停留在前方，表情中性，呈现出继续观察或准备离开的自然状态。

# Constraints
- 顾客必须始终正面可见
- 面部无遮挡，五官清晰
- 不能出现任何商品、货架、价格标签、机器本体或屏幕 UI
- 无其他人介入交互
- 无夸张表情、无夸张动作、无表演感
- 整体效果像真实零售设备前置摄像头记录到的顾客浏览片段
